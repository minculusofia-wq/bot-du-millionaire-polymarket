# Bot du Millionnaire - Solana Copy Trading

## Overview
A Solana copy trading bot with a modern web interface. The bot monitors and tracks Solana trader wallets in real-time, allowing users to mirror trades from successful traders automatically.

## Project Type
- **Language**: Python 3.11
- **Framework**: Flask web application
- **Blockchain**: Solana
- **Purpose**: Automated copy trading bot with TEST and LIVE modes

## Current State
- ✅ Flask web server running on port 5000
- ✅ Web interface with 4 tabs: Dashboard, Trader Management, Settings, History
- ✅ TEST mode with real portfolio tracking via Helius RPC
- ✅ 10 configurable traders with max 3 active simultaneously
- ✅ Real-time performance tracking and PnL calculations
- ✅ Security: Private keys stored only in session memory

## Project Architecture

### Main Files
- `bot.py` - Flask application with HTML template and API routes
- `bot_logic.py` - Backend logic for bot control and trader management
- `portfolio_tracker.py` - Real-time wallet tracking via Helius RPC
- `config.json` - Configuration including traders, settings, and RPC URL
- `portfolio_tracker.json` - Tracked wallet data with PnL
- `blockchain_monitor.py` - Blockchain monitoring utilities
- `simulation_engine.py` - Trading simulation engine
- `test_engine.py` - Testing utilities

### Key Features
1. **Web Interface**: Modern dark theme, responsive design
2. **Trader Management**: Edit trader names, emojis, addresses in real-time
3. **Portfolio Tracking**: Monitors real Solana wallets every 30 seconds
4. **Performance Metrics**: PnL tracking, success rate, best trader identification
5. **Security**: Private keys never saved to disk in plain text

## Dependencies
- Flask - Web framework
- requests - HTTP library for RPC calls

None documented yet.
